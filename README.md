# MBFPorter

MBFPorterは、Minecraftの生物（Mob）を特定の場所に一瞬でテレポートさせるためのBukkitプラグインです。Folia対応で、マルチスレッド環境でも安全に動作します。

## 機能

- **転送スクロール**: 特殊なアイテムでMobをクリックすると、設定された座標にテレポートさせます
- **WorldGuard連携**: 保護されたエリア内のMobには使用できないようにする保護機能
- **Folia対応**: 最新のFoliaサーバー環境で最適化された動作

## 要件

- Bukkit/Spigot/Paper 1.20.x 以上
- WorldGuard プラグイン
- （オプション）Folia - Folia環境での実行をサポート

## インストール方法

1. このプラグインの`.jar`ファイルをサーバーの`plugins`ディレクトリに配置します
2. サーバーを再起動または`/reload`コマンドを実行します
3. WorldGuardがインストールされていることを確認してください

## 使い方

### コマンド

- `/getwand` - 転送スクロールを入手します（権限: `mbfporter.getwand`）

### 転送スクロールの使用方法

1. `/getwand`コマンドで転送スクロールを入手します
2. テレポートさせたいMob（動物、モンスターなど）に向かって右クリックします
3. Mobは即座に設定された座標（ワールド"world"の座標 X:4949, Y:68, Z:-1149）にテレポートします
4. 転送スクロールは1回の使用で消費されます

### 注意事項

- WorldGuardで保護されたエリア内にいるMobには使用できません
- プレイヤーをテレポートすることはできません
- メインハンドに持って右クリックする必要があります

## 権限

- `mbfporter.getwand` - 転送スクロールを入手するためのコマンド権限（デフォルト: OP）

## 設定

現在、設定ファイルはありません。テレポート先の座標はプラグイン内にハードコードされています（`world`ワールドの X:4949, Y:68, Z:-1149）。
